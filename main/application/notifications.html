About us

description of the application

E-Psychotherapist (A web application for psychoanalysis and diagnosis) is about diagnosed and analyze the mental condition of a patient using the various test as well as emotion detection algorithms. However, the primary motive of this application is to diagnose a mental illness through the standard approaches; this application is just for giving preliminary information about the possible psychological disorder to the patients and their psychiatrists.
